# English4Ever - Flashcard Learning Platform

## Project Overview

English4Ever is a web-based flashcard application for learning English, inspired by Memrise's clean and focused learning experience.

## Project Status

🚧 **In Development** - Initial setup phase

## Tech Stack

To be determined. Recommended options:
- **Modern**: React with Tailwind CSS, Backend: Java Spring Boot, DB: MongoDB


## Key Features (Planned)

Based on Memrise-style learning:
- Focused flashcard interface with minimal distractions
- Progress tracking during learning sessions
- Multiple choice and text input answer modes
- Spaced repetition algorithm by Spaced Repetition System (Hệ thống ôn tập ngắt quãng - SRS). Đây là cốt lõi của các ứng dụng học ngôn ngữ nổi tiếng như Anki hay Quizlet.
- Clean, centered card design
- Mobile-responsive layout

## Getting Started

1. Choose tech stack(Reactjs(folder englishever-fe), Spring Boot, mongoDB)
2. Initialize project
3. Implement flashcard UI based on design mockups(folder images)
4. Features detail:
   1. Create course for all users
   2. Create unit for course
   3. Create flashcard for unit
   4. Each unit has: flashcard, learn(by word by word: type or choose answer), match word (like images folder)
5. Implement learning algorithm SRS (for learning)

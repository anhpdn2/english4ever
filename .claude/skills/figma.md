# Figma Design Integration

Use this skill to work with Figma designs for the English4Ever project.

## Purpose

Fetch and analyze Figma design mockups to implement UI components accurately.

## Instructions

When invoked:
1. Get basic info about the Figma file
2. Retrieve screenshots of relevant artboards
3. Analyze the design structure
4. Provide implementation guidance based on the design

## Usage

Invoke with: `/figma [figma-url]`
